📌 Sistema de Gestão de Encomendas para Costura
📖 Sobre o Projeto

Este projeto foi desenvolvido para a disciplina de Programação Orientada a Objetos, com o objetivo de automatizar o processo de gestão de encomendas de uma empreendedora do ramo de corte e costura.

Atualmente, o controle é feito manualmente, o que gera dificuldades na organização e no acompanhamento das entregas. A proposta do sistema é centralizar essas informações e automatizar a comunicação com os clientes.

🎯 Objetivo

Desenvolver um sistema integrado capaz de:

Gerenciar encomendas (entrada e saída de peças)
Armazenar dados de clientes e serviços
Controlar prazos de entrega
Automatizar o envio de mensagens para clientes via WhatsApp
🧩 Arquitetura do Sistema

🛠️ Tecnologias Utilizadas

🐍 Python
📱 Flutter (Dart SDK)
🐘 PostgreSQL
🐳 Docker
💻 Visual Studio Code
🔗 API REST

⚙️ Funcionalidades

Cadastro de clientes e encomendas
Registro de serviços (ex: barra de calça, ajustes, etc.)
Controle de valores e datas de entrega
Listagem de entregas do dia
Automação de notificações via WhatsApp
